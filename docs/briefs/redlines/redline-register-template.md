## How to use this file

- One row per redline.
- Append new rows at the end; do not rewrite existing rows.
- Status transitions: `Open` → `Resolved` (or `Withdrawn`).
- A `Resolved` entry remains in the register as historical record.
- The Decision column should either resolve the issue inline (clarification text) OR point to where the answer lives (chat session, ChatGPT review record, addendum entry, etc.).
