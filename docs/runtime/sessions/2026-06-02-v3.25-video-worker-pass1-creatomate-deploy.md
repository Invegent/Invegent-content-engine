## Lessons captured this session

- **L-v3.25-a (operational; STRONG) — `safe-deploy` reads the EF-drift register (`m.vw_ef_drift_current`), NOT a live recompute.** A merge to main does not refresh it; the cached snapshot must be recomputed. **In-design path for a feature-branch EF deploy:** (1) merge the verified branch to main; (2) refresh the **targeted** EF-drift register (`drift-check?write=true&slug=<ef>`); (3) confirm **B-FD / pending deploy**; (4) deploy with `--allow-warn`; (5) refresh drift again; (6) confirm **A-LE / clean**. **Do not bypass the wrapper** (no raw `supabase functions deploy`); class A/A-LE is a hard block with no override, B-FD/B-RR pass only with `--allow-warn`.
- **L-v3.25-b — Creatomate `volume` is a percentage string contract.** Numeric `1.0` is read as 1% (−40 dB). Use `"100%"`/`"15%"`. Verify audio renders by **measuring loudness** (ffmpeg `volumedetect`/`ebur128`), not by ear alone — CCB could confirm stream presence but not level; the −40 dB drop was only caught by measurement + human listening together.
